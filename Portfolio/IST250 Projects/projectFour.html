Creating a well-styled website

Assignment Description

  • Create three HTML pages. Each page should have a purpose within the context of your overall site.
  • Create at least one CSS file that provides styling for all three of your HTML pages.
  • One of your pages should have a floating element that includes both text and an image. It should have proper margins. Other text should wrap around that floating element.
  • Use Google Fonts.
  • Use span and div as shown in class.
  • Modify background and foreground colors of at least one element using hexadecimal values from CSS.
  • Apply a CSS style to an entire web page using a single CSS selector.
  • Use a CSS class selector to apply a style to multiple page elements.
  • Use a CSS hover selector to make something happen when the user places their mouse pointer over an element.
